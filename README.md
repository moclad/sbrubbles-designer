# sbrubbles-designer

Report Template Designer - A visual designer for Microsoft RDL (Report Definition Language) 2008 reports.

## Project Structure

- **designer-app/** - The main React + TypeScript application

## Quick Start

```bash
cd designer-app
npm install
npm run dev
```

For detailed information, see the [designer-app README](./designer-app/README.md).

## Features

- Visual drag-and-drop report designer
- Plugin-based report items (TextBox, Table, Image, QRCode, etc.)
- RDL 2008 XML export
- Template persistence (save/load as JSON)
- Alignment and distribution tools
- Keyboard shortcuts for efficient editing
